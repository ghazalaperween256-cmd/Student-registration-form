
# 🎓 Student Registration Form

A modern, responsive, and visually appealing Student Registration Form built from scratch using front-end web technologies.

## 🚀 Features
- **Modern UI:** Clean card layout with a beautiful purple gradient background.
- **Responsive Design:** Works perfectly on both desktops, tablets, and mobile screens.
- **Form Elements:** Includes text fields, email/phone formats, date picker, radio buttons, and custom dropdowns.
- **Interactive Feedback:** Visual success message changes on clicking the "Register Now" button.

## 🛠️ Technologies Used
- **HTML5:** For creating the clean structure of the form.
- **CSS3:** For custom styling, Flexbox, Grid layout, and Gradients.
- **JavaScript:** For basic input validation and handling button events.

## ⚙️ How to Run this Project Locally
1. Download or clone this repository.
2. Open the `register.html` file in any modern web browser (Google Chrome, Brave, Edge, etc.).
